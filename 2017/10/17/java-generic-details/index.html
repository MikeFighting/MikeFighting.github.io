Java泛型中需要注意的问题

  1. 不能使用基本数据类型初始化泛型参数
  2. Runtime类型检查不起作用
  3. 不能使用参数类型来创建Array
  4. 可变参数警告
  5. 不能初始化类型变量
  6. 泛型类中的静态环境中不允许使用类型变量
  7. 不能抛出或者捕获泛型类对象
  8. 注意泛型擦除之后的冲突

其它

Array有特殊的保护，不能对存有父类的Array中添加子类元素，否则在运行时虚拟机会抛出ArrayStoreException的异常。
